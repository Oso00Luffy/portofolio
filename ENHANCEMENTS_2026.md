## 🎨 Customization Guide

### Change Primary Colors
Edit the CSS variables at the top of `style.css`:
```css
:root {
  --primary-gradient: linear-gradient(135deg, #YOUR_COLOR_1, #YOUR_COLOR_2);
}
```

### Adjust Animation Speed
Change timing in animation definitions:
```css
@keyframes fadeInUp {
  /* Adjust duration */
  animation: fadeInUp 0.6s ease; /* was 0.8s */
}
```

### Modify Glassmorphism Effect
Adjust blur amount:
```css
backdrop-filter: blur(25px); /* was 20px */
```
